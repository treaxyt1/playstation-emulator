Common Issues & Solutions

üéÆ Game Won't Load

  • Ensure your ROM file is in .BIN format (not .ISO or .IMG)
  • Check that the file size is between 200-700MB (typical PS1 game size)
  • Try a different ROM file - your current file may be corrupted
  • Clear your browser cache and reload the page
  • Make sure you have both .BIN and .CUE files if the game requires them

‚ö° Performance Issues / Lag

  • Enable hardware acceleration in your browser settings
  • Close unnecessary browser tabs to free up RAM
  • Disable browser extensions that might interfere
  • Use Chrome or Edge for best performance
  • Lower graphics settings in the emulator if available
  • Ensure your device meets minimum requirements (4GB RAM, modern CPU)

üéß Audio Crackling or No Sound

  • Increase the audio buffer size in emulator settings
  • Close other applications using audio
  • Check your browser's audio permissions
  • Try refreshing the page
  • Ensure your system volume is not muted

üéØ Controller Not Working

  • Check that your browser has gamepad permissions enabled
  • Unplug and reconnect your controller
  • Try a different USB port
  • For Bluetooth controllers, ensure they're properly paired
  • Use keyboard controls as a fallback (see controls in the help modal)
  • On mobile, the touch gamepad should appear automatically

üíæ Save States Not Working

  • Ensure your browser allows local storage
  • Check that you're not in private/incognito mode
  • Clear browser cache and try again
  • Make sure you have enough storage space
  • Try using in-game saves instead of save states

üì± Mobile Gamepad Issues

  • Ensure your screen width is below 768px for gamepad to appear
  • Try rotating your device to landscape mode
  • Adjust gamepad opacity and size in settings (‚öôÔ∏è button)
  • Enable haptic feedback in gamepad settings if supported
  • Refresh the page if gamepad doesn't appear

üåê Browser Compatibility

  • Best: Chrome 100+, Edge 100+, Brave 1.40+
  • Good: Firefox 100+, Safari 15+
  • Mobile: Chrome/Safari on iOS 13+, Android 8+
  • Update your browser to the latest version for best results

System Requirements

Minimum

  • Modern browser (Chrome, Firefox, Edge, Safari)
  • 4GB RAM
  • Dual-core CPU (2GHz+)
  • WebGL 2.0 support
  • Stable internet connection

Recommended

  • Chrome or Edge (latest version)
  • 8GB+ RAM
  • Quad-core CPU (2.5GHz+)
  • Dedicated GPU
  • Hardware acceleration enabled
