Quick Help

Before Contacting Support

Please check our FAQ page and user guide first. Most common questions are answered there.

Bug Reports

If you've found a bug, please report it on our GitHub Issues page with details about your browser, operating system, and steps to reproduce.

ROM Support

We cannot provide ROM files or help with obtaining them. Please see our ROM resources page for legal information and trusted sources.

Response Time

We typically respond within 24-48 hours on Telegram. GitHub issues may take longer depending on complexity.

Support the Project

ePSXemu is a free, open-source project dedicated to game preservation. If you'd like to support development, consider:

  • Sharing ePSXemu with other retro gaming enthusiasts
  • Contributing code or documentation on GitHub
  • Reporting bugs and suggesting improvements
  • Spreading awareness about legal game preservation
